Welcome to the Corespeed API documentation!

**Corespeed** is a platform that help you deploy your service with one click, No matter what programming language you use, what framework you use.

You can join our Discord Server to learn more and get a priority support!

## Advantages of Corespeed

- Convenient and Fast
- No need to worry about the server
- No need to worry about the infrastructure
- No need to worry about the deployment
- No need to worry about the scaling
- No need to worry about the monitoring
